PSP-Superhelden
===============
## Requirements:
Node.js, NPM, ruby & compass (optional)

## Installation:
Projekt auschecken, in den Ornder wechseln<br>
```$ npm install```<br>
```$ bower install```<br>
und los gehts ... <br>
Server starten mit:<br>
```$ grunt serve````<br>
Build starten mit:<br>
```$ grunt build```<br>
